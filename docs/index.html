一次蜕变，
两个世界。

全新的浅色设计语言、50+ 设计令牌、代理宏架构、3 项关键 Bug 修复——V2.3 是 ClipWheel 有史以来最大的视觉与架构升级。

核心特性

双主题系统：浅色与暗色，一键切换

基于 Claude.com 暖调编辑风格设计的全新浅色 UI——奶油底色搭配珊瑚强调色，温暖而不失专业。同时保留经典暗色主题，通过设置面板的勾选框即时切换，重启后自动持久化。

架构升级

TC_* 代理宏设计系统

50+ 色值常量，覆盖背景层、强调色、文字层级、边框状态、轮盘色板、取消区域等全部 UI 元素。所有绘制代码统一使用 TC_* 代理宏，根据全局主题自动选择明暗值，无需手动 if/else。

新特性

主题感知字体

浅色模式使用 Georgia（标题）搭配 Segoe UI（正文），营造温暖的阅读氛围；暗色模式保持 Microsoft YaHei UI，确保清晰度与一致性。5 级字号常量统一管理。

Bug 修复

3 项关键修复

  • 卡死修复：轮盘预览中点击固定扇区未拖拽导致鼠标永久捕获的问题
  • 取消固定错位：X 按钮与内容不匹配，现在正确映射到扇区索引
  • 高扇区不可交互：扇区 5-7 的固定项现在可正常显示 X 按钮和标签
  • 对比度合规：浅色主题 muted 文字从 3.95:1 提升到 4.8:1（WCAG AA）
重构

代码质量提升

  • 函数拆分：paint_wheel_preview() 拆分为 5 个静态绘制函数
  • 字体常量化：FONT_SIZE_DISPLAY / TITLE / BODY / RENAME / CAPTION
  • 刷子重命名：g_brush_light → g_brush_bg_deep 等语义化命名
  • 色值扩展：新增 17 个浅色主题变体常量（取消区域、X 按钮、标签）
核心功能

为速度而生，
为直觉而设计。

每一个交互细节都经过深思熟虑，从径向轮盘到控制中心，ClipWheel 让你与剪贴板之间只剩一个动作。

径向轮盘

热键一按，轮盘即刻在鼠标处弹出。通过方向感知精准选取历史片段，松开热键即自动粘贴——完全不需要眼睛看屏幕。

智能热键系统

任意组合修饰键，在 UI 中点击即录制。无需重启，设置实时生效。

使用流程

四步，一气呵成。

01

启动

运行 clipwheel.exe，自动最小化至系统托盘，静默守候。

02

复制

在任何应用中正常使用 Ctrl+C，ClipWheel 在后台默默记录一切。

03

呼出

按住热键（默认 ~），轮盘弹出，移动鼠标指向目标片段。

04

释放

松开热键，内容瞬间填入光标所在位置，心流不中断。
